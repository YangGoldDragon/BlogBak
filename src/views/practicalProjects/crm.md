# Vite + Vue3 + Ts开发Crm系统

## 技术栈:
- Vite4.x
- Vue3.x
- Typescript4.x
- Vant4.x

## 使用vite创建项目

- 首先，我们通过阅读vite官网（https://cn.vitejs.dev/guide/） 创建初始项目，
我这里使用的是pnpm安装的方式。按照以下命令运行：

```typescript
pnpm create vite
project name: web-h5
Vue
Typescript
cd web-h5
pnpm install
pnpm run dev
```
